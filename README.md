**Functional Requirement :**

**USER**

1. As a user when i open Bike rental app, i should see list of rental bikes available near my location.

2. User should be able to select the pickup time and time as well as drop time and date.

3. After the selecting the date and time, user should see the quoted amount to be paid.

4. User should be able to verify this identity with Driving Licence or AadharCard.

5. App login can be done with AadharCard linked number or Login with Google.

6. Once verification is done, the user should move to payment screen. Once payment is done user should have a confirmation ID.

**ADMIN**

7. As a admin when i open Bike rental app, i should see the list of bikes added by Same admin.

8. When User comes to collect the bike, user should show his original driving licence and get reverified with Admin.

9. Admin should enter the current bike status ex: kilometres, fuel points.

10. Once the user and bike verified, Admin will hand over the Bike keys to the User.

11. Admin can see wether the bike is returned or not.

12. When User returning the bike only admin can close the ride the by verifying the bike damage, with in given time or not, Bike should have atleast one fuel point.
